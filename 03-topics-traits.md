# נושאים/יעדים ותכונות אופי להערכת המועמדים

## תשעת הנושאים (נבחרו לפי מצע המפלגה, סדר היום הציבורי והתמהיל בפועל של 51 המועמדים)

| # | נושא | מה נמדד |
|---|------|---------|
| T1 | **ביטחון, סיום המלחמה והשבת החטופים** | עמדות ופעילות בסוגיות צבא וביטחון, מאבק החטופים, יום שאחרי |
| T2 | **דמוקרטיה, משפט ושלטון החוק** | מאבק בהפיכה המשטרית, עצמאות בתי המשפט, זכויות אדם ואזרח |
| T3 | **דת ומדינה ושוויון בנטל** | גיוס חרדים, חופש דת, פלורליזם יהודי, כפייה דתית |
| T4 | **הסדרה מדינית ושלום** | שתי מדינות, יחסים אזוריים, דיפלומטיה |
| T5 | **שוויון אזרחי ושותפות יהודית־ערבית** | החברה הערבית/דרוזית/צ'רקסית, מאבק בגזענות, פריפריה גיאוגרפית וחברתית |
| T6 | **כלכלה, עבודה ורווחה** | יוקר מחיה, דיור, זכויות עובדים, מדינת רווחה |
| T7 | **חינוך ובריאות ציבוריים** | מערכות הציבוריות, שוויון הזדמנויות |
| T8 | **סביבה ואקלים** | אנרגיה, שטחים פתוחים, משבר האקלים |
| T9 | **שקיפות, טוהר מידות ותיקון השלטון** | מאבק בשחיתות, שקיפות שלטונית, ממשל תקין |
| T10 | **שוויון מגדרי ולהט"ב** | מעמד האישה, אלימות נגד נשים, זכויות הקהילה הגאה |

*(10 נושאים — בגבול העליון שהוגדר, כי תמהיל המועמדים מגוון מאוד)*

### חמשת ממדי הדירוג לכל נושא (סולם 1–5)
| ממד | תוכן | מקורות קבילים עיקריים |
|-----|------|------------------------|
| D1 | חשיבות שהמועמד מייחס לנושא — הצהרות והבטחות | אתר מפלגה/מועמד (כאן עדות עצמית קבילה) |
| D2 | מעורבות בהתבטאות בכלי התקשורת | ראיונות, טורים, סיקור |
| D3 | מעורבות בפעולות בשטח | ארגונים שהקים/ניהל, מחאות, חקיקה, תפקידים |
| D4 | הצלחה / שותפות בהצלחה בהשגת יעדים | תוצאות מתועדות: חוקים שעברו, ארגונים פעילים, הישגים |
| D5 | קרדיט מהאחרים | תמיכות, פרסים, הכרה ציבורית, קרדיט ממתחרים |

## חמש תכונות אופי

| # | תכונה | מה נמדד |
|---|-------|---------|
| C1 | **יושרה וטוהר מידות** | היעדר כתמים, עקביות בין הצהרות למעשים, שקיפות אישית |
| C2 | **חריצות וביצועיזם** | הספק מוכח, התמדה רב־שנתית, יכולת להפוך כוונה לתוצאה |
| C3 | **אומץ ציבורי** | נכונות לעמוד מול לחץ, לשלם מחיר אישי, להתעמת עם בעלי כוח |
| C4 | **יכולת חיבור ועבודה רוחבית** | בניית קואליציות, גישור בין מגזרים, עבודת צוות |
| C5 | **מקצועיות וניסיון רלוונטי** | עומק מקצועי בתחום מומחיות + ניסיון ניהולי/פרלמנטרי |

### ארבעת ממדי הדירוג לכל תכונה (סולם 1–5)
| ממד | תוכן |
|-----|------|
| E1 | עדות המועמד על עצמו (הצהרות, הבטחות) |
| E2 | קרדיט ממועמדים אחרים / עמיתים |
| E3 | כפי שמשתקף מהתקשורת (סיקור, תחקירים, פרופילים) |
| E4 | רלוונטיות התכונה לפעולות בשטח שהמועמד מעורב בהן (עבר והווה) |

## כלל דירוג בתנאי מידע חסר
- מועמד ללא ראיות חיצוניות בנושא: D1 נקבע לפי הצהרותיו (קביל), D2–D5 מקבלים 1–2 לכל היותר, וציון הביסוס (Evidence) של המועמד יורד.
- ציון ביסוס לכל מועמד (1–5): 5 = רקורד ציבורי מתועד היטב (ח"כ עם ויקיפדיה + סיקור), 1 = עדות עצמית בלבד.
