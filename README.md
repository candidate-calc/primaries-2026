## איך מגיעים להחלטה (מסלול מומלץ)
1. פתחו את primaries-scoring.xlsx ← גיליון **פרמטרים**: תנו משקל 8–10 לנושאים שהכי חשובים לכם, 0–2 לשוליים; כוונו משקלי תכונות ו־β.
2. גיליון **דירוג סופי** — קראו את 12 הראשונים.
3. לכל מועמד/ת בצמרת — פתחו את התיק ב־candidates/ וקראו את הדגלים (🟢/🔴) והראיות.
4. בצעו ניתוח רגישות (ראו 04) וגבשו שמינייה יציבה.
5. שקלול אחרון: שריונים (מגדר/מרצ/מיעוטים/כפרי) — ייתכן שתרצו לתעדף מועמד שנאבק על מקום פתוח על פני מי שמובטח לו שריון.
